BEHAVIOR:

	Join a experiment >
		(only currently open studies)
		Studies[] (First time, no cookie) >
			"Privide your information"
			{
				Experiment setup tester data
			}
			Continue > (give cookie) V
		Studies[] (with cookie) >
			"%name%, give a rating"
			Element Identifiers [] >
				{
					rating
					comments
				}
				submit ^
	Manage a experiment >
		Studies[] >
			Advance stage >
		New Experiment >
			"Provide information about the experiment"
			{
				Name
				Info to know about each element
					{
						Name*
						Price
						Etc..
					}
				Info to know about each tester
					{
						Name*
						Experience
						Comments
						Etc..
					}
			}
			Continue >
				"Add test elements"
				{
					Data (fields from previous screen)
				} []
				* New test element
				Continue to first mapping (warning) >
					"Give each element an arbitrary identifier"
					Test Element Name = ""[]
					Continue to second mapping >
						"Scramble the identifiers and note the reordering"
						%identifier% = ""[]
						Begin Experiment >
							"Currently Accepting Reviews"
							(becomes available for joining)
							Close Experiment >
								"Experiment is concluded"
								Export results >
				
			
			
			
STAGES OF A EXPERIMENT:
	- Setup
		list all elements, questions for participants
	- Map stage 1
		arbitrary new identifiers
	- Map stage 2
		Scramble identifiers
	- Open experiment
		accept reviews
	- Close experiment, export data
		reverse mappings
